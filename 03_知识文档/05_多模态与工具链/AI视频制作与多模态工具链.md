# 📚 AI 视频制作与多模态工具链

## 概述

从 AI + 实拍合成的视频制作工作流，涵盖首帧生成、多模态替换、动作迁移等核心技术。包括社区一线创作者的工具选型和实操经验。

## 核心内容

### AI+实拍视频制作流程

**步骤一：首帧生成**
- 工具：香蕉（banana）、即梦、千问
- 目的：为后续画面生成基础帧
- 技巧：提示词描述画面构图、光影和风格

**步骤二：多模态替换/动作迁移**
- 替换画面元素：Wan、Kling
- 动作迁移：Runway
- 风格统一：保持各帧之间的视觉连贯性

### 工具链详解

**ComfyUI 生态**
- 曾经最流行的节点式工作流工具
- 现在使用率下降，但仍是重要的基础工具
- 优势：高度可定制、社区节点丰富
- 劣势：上手门槛高、版本兼容问题多

**当前推荐入门学习方向**
| 工具 | 用途 | 难度 |
|------|------|------|
| 香蕉 (banana) | 首帧/图片生成 | ⭐⭐ |
| image2 / MJ | 文生图/图生图 | ⭐⭐ |
| Seedance 2.0 | 视频生成 | ⭐⭐⭐ |
| 可灵 (Kling) | 视频生成/动作迁移 | ⭐⭐⭐ |
| Wan | 多模态视频处理 | ⭐⭐⭐⭐ |
| Runway | 专业视频编辑+AI | ⭐⭐⭐⭐ |

### 技能获取路径
- 基础操作可通过网上教程自学
- 更高级的工作流需要自己研究尝试
- 推荐关注内容创作者的实操分享而非教科书式教程

## 最佳实践

- AI+实拍结合效果最好，纯 AI 生成的视频缺乏真实感
- 首帧质量决定最终效果，花 80% 精力在首帧
- ComfyUI 虽然学习曲线陡，但定制能力无可替代
- 多模态工具选型跟随社区趋势，保持关注新工具
