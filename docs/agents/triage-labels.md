# Triage Labels

| Role | Label |
|------|-------|
| 需要评估 | `待评估` |
| 等待回复 | `待回复` |
| AI 可处理 | `AI可处理` |
| 需人工处理 | `需人工` |
| 不处理 | `不处理` |

## Usage

These labels are applied by the `triage` skill as issues move through the triage state machine. They must exist in the GitHub repository before use — create them via `gh label create "<name>"` or the GitHub UI.
