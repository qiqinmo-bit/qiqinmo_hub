# Issue Tracker — GitHub

Issues are tracked on **GitHub Issues** for the repo `qiqinmo-bit/qiqinmo_hub`.

## Commands

- **Create issue**: `gh issue create --title "<title>" --body "<body>"`
- **List issues**: `gh issue list`
- **View issue**: `gh issue view <number>`
- **Close issue**: `gh issue close <number>`
- **Reopen issue**: `gh issue reopen <number>`
- **Add label**: `gh issue edit <number> --add-label "<label>"`
- **Remove label**: `gh issue edit <number> --remove-label "<label>"`

## URL format

`https://github.com/qiqinmo-bit/qiqinmo_hub/issues/<number>`
